# GenesisW Bot

Бот для поиска каналов в Telegram.

## Команды
- `/start` - информация
- `/search` - поиск каналов
- `/premium` - премиум доступ
- `/admin` - админ панель
- `/help` - справка
- `/test` - тест поиска
